# Omi3.0 小红书笔记生产线

> 一套适用于任何 Agent 平台的小红书图文笔记自动生成 Skill。输入内容素材 + IP形象，自动产出完整的分镜脚本 JSON + 封面图 + 内容页图。

**支持平台：** Claude Code · OpenClaw · Codex · 扣子（Coze）· 任何具备本地图片生成能力的 Agent

---

## 这是什么

Omi3.0 是一个与平台无关的 **Agent Skill**，核心是一套经过打磨的 Prompt 流水线，只要你的 Agent 具备图片生成能力，就能直接运行。

```
输入：原始素材文本 + IP形象图 + 风格需求
  ↓
[Step 1] 资产生成器     → 品牌视觉资产看板（16:9）
[Step 2] 笔记导演       → 分镜脚本 JSON（纯 LLM，无需图片能力）
[Step 3] 封面生成大师   → 封面图（3:4，4K）
[Step 4] 内容生成大师   → 内容页图（3:4，2K，多张）
```

- **Step 2** 是纯 LLM 任务，任何 Agent 均可执行
- **Step 1 / 3 / 4** 需要图片生成能力（图生图）

```
    ├── 01-资产生成器.md    # 品牌视觉资产看板生成 Prompt（图生图，16:9）

    └── 04-内容生成大师.md  # 内容页图生成 Prompt（图生图，3:4，2K）
```

## 安装方法

### Claude Code

直接把下面这句话发给 Claude Code，让它帮你完成安装：

```
帮我安装这个 skill：git clone https://github.com/CzzzzzzJ/omi.git ~/.claude/skills/omi
```

重启 Claude Code 后，输入 `/omi` 即可触发。

### 其他 Agent 平台（OpenClaw / Codex / 扣子等）

直接把下面这句话发给你的 Agent，让它帮你完成安装：

```
帮我安装这个 skill：git clone https://github.com/CzzzzzzJ/omi.git
prompts/ 目录中的文件作为各步骤的 Prompt 引用
```

## 使用方式

```
/omi <内容素材> [--need <需求描述>]
```

**示例：**

```
/omi Claude Code完全指南 --need 面向小白，突出省时省力
/omi GLM-5测评报告 --need 科技感风格，重点展示速度对比数据
/omi 飞书多维表格实战技巧
```

**参数说明：**
- `内容素材`：直接粘贴文章 / 报告文本，或描述主题
- `--need`：风格偏好、目标受众、内容侧重（可选）

## 前置要求

| 要求 | 说明 |
|------|------|
| LLM 能力 | 任何 Agent 平台均满足 |
| 图片生成能力 | 需接入图生图工具（Gemini Image、DALL·E、Stable Diffusion 等） |
| IP 形象图 | 首次使用时提供，后续复用资产图即可 |

> 如果当前环境没有图片生成能力，Skill 会提示你补充能力后重试，但 Step 2（分镜脚本）仍可正常输出。

## 步骤详解

### Step 1：资产生成器

生成「品牌视觉资产看板」，作为后续所有图片的**唯一视觉参考源**：
- IP 形象标准立绘
- 5-6 色色卡（主色 60% / 辅助色 30% / 点缀色 10%）
- 字体层级示例（标题字形 + 正文纹理）
- 封面 / 内页版式蓝图（线框图风格）
- 风格化装饰 UI 组件

首次使用时生成一次，后续直接复用。

### Step 2：笔记导演（纯 LLM）

将原始素材重构为结构化分镜脚本 JSON，自动拆分输出：
- `cover_json`：封面页数据
- `content_json`：内容页数据列表（3-5 页）

**封面**：IP 是主角（Hero Mode），大字报排版，抢夺注意力。
**内页**：IP 是配角（Guide Mode），内容为王，IP 占比 < 15%。

### Step 3：封面生成大师

- 文字占画面 40-50%，极大化排版
- IP 与标题发生互动（穿插、抱道具、被压住等）
- 严格复用资产图的配色方案

### Step 4：内容生成大师

- 卡片容器 + 项目符号列表 + 金句高亮
- IP 缩在角落，手持指示牌或指向文字
- 严格执行分镜脚本中的 `visual_layer` 布局指令
- 多页时逐页传入，分别生成

## 内容规范（自动执行）

| 规则 | 说明 |
|------|------|
| 违禁词 | 禁止「极 / 最 / 第一 / 首个」等绝对化表达 |
| 外链 | 禁止明示外部链接或引导站外 |
| 福利页 | 禁止单独福利领取页面 |
| 敏感词 | 禁止翻墙 / 梯子相关字样 |
